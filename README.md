Aim:
Responsive banking interface with deposit and withdrawal functionality

Learning outcomes:
1. Understand the structure of a basic web application using HTML for layout, CSS for styling, and JavaScript for functionality.
2. Implement client-side interactivity using JavaScript functions and event handling (button clicks). 
3. Integrate multiple web technologies (HTML, CSS, JS) into a single working application.
4. Gain practical exposure to real-world scenarios such as balance management in a banking system.
